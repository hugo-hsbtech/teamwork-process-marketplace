# Grounding — calibrate against the bundled exemplar

Quality is judged against a **self-contained golden exemplar** shipped with the
skill: [`../assets/golden-example.md`](../assets/golden-example.md). It needs no
external repository — the skill works in a fresh project. Read it before declaring
a run done and compare your output for the same *quality bar* (not the same
content).

## The calibration checklist

**Problem = pain, not solution.** The exemplar states the pain concretely with
observable symptoms and never prescribes a solution. Thin output names a feature;
good output names the pain the feature would relieve.

**Confidence is honest and mid-range.** The exemplar is not a wall of 95s — it
runs the well-evidenced sections high and the estimated ones lower (e.g. a
business-impact figure marked "estimate, not calculated"), each soft section
carrying a `hint` that says what would raise it.

**Dispositions are used, not avoided.** Blocking sections can clear the gate via
honest `assumption`/`discovery` and the demand is still ready. "We don't know yet,
here's the plan" is readiness, not failure.

**Tensions are registered with resolutions**, not silently dropped.

**Derived sections are defensible and flagged.** Where the template has a `derived`
section (e.g. a triage draft), every claim traces to captured evidence and the
section is marked as a draft pending human sign-off — it never pretends to be a
decision a human hasn't made. See the template's companion guide.

## Quick self-check before finishing

- [ ] Does every blocking section describe substance, not a guess (or carry an honest disposition)?
- [ ] Does every captured/derived section carry its `Confidence/Source/Status/Disposition/Hint` line?
- [ ] Are confidences honest and mid-range where evidence is soft, each with a real hint?
- [ ] Are assumptions/discovery items explicit, with owners or a time-box?
- [ ] Are derived sections flagged as drafts where human judgment is required?
- [ ] Does it read like the exemplar — a demand *understood* — not a thin form?

If any box is unchecked, you are not done: ask the missing question or route the
gap to an honest disposition.
